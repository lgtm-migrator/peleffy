Solvent models¶

This module provides different methods to parameterize solvent models for PELE.

Primary objects¶

OBC2

Implementation of the OBC2 solvent.

OPLSOBC

It defines a template generator for OBC compatible with the OPLS2005 force field implemented in PELE.

Secondary objects¶

OBC1

Implementation of the OBC1 solvent.
